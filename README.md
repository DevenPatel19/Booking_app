# User Stories for Hotel Booking App

## User Authentication and Authorization-

- As a user, I want to be able to create an account so that I can log in and use the hotel booking app.

- As a user, I want to log in with my email and password to access my account.

- As a user, I want to be able to reset my password in case I forget it.

- As an administrator, I want to have special access to manage user accounts and permissions.

## Managing User Profile-

- As a user, I want to be able to view and edit my profile information.

## Hotel Listings-

- As a user, I want to see a list of available hotels so that I can browse and select one for booking.

- As a user, I want to view detailed information about a specific hotel, including its name, location, description, and amenities.

- As a user, I want to be able to search for hotels based on location, price, and other criteria.

## Booking Rooms-

- As a user, I want to book a room in a hotel by selecting check-in and check-out dates and specifying the number of guests.

- As a user, I want to see the total cost of my booking before confirming it.

- As a user, I want to receive a confirmation of my booking and see it in my profile.

## Reviews and Ratings (Optional)-

- As a user, I want to read reviews and ratings of hotels from other users to help me make booking decisions.

- As a user, I want to leave a review and rating for a hotel after my stay.

## Payment Processing-

- As a user, I want to be able to make payments securely for my bookings using different payment methods.

- As a user, I want to see a payment confirmation and receipt after completing a booking.

## Responsive Design-

- As a user, I want the app to work seamlessly on both desktop and mobile devices for a consistent user experience.
